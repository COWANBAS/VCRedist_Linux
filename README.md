# Como instalar

Com o wine ja instalado baixe os arquivos no release e extraia, clique com o botão direito no script *Instalar.sh* de permissão de executar como um programa depois disso basta executar o arquivo que ele vai começar a baixar todas as extensão que estão na pasta.
